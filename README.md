## ✨ Key Features

### 🏢 **For Administrators**
*   📊 **Real-time Insights**: Track student participation and revenue live.
*   📅 **Dynamic Event Hub**: Create, edit, and manage events effortlessly.
*   🤝 **Coordinator Control**: Assign and manage event leads.
*   📜 **One-Click Certification**: Generate and email PDF certificates instantly.
*   📥 **Data Portability**: Export critical data to CSV formats.

### 🧑‍🎓 **For Students**
*   🎟️ **Event Discovery**: Explore a vibrant list of campus activities.
*   💳 **Easy Registration**: Securely join events with automated payment records.
*   📁 **My Dashboard**: View and manage all your event participations.
*   📥 **Direct Downloads**: Get your certificates directly from your profile.
*   📧 **Smart Notifications**: Stay updated with automated email alerts.

## 🚀 Getting Started

### 📝 Prerequisites
*   ☕ **JDK 17**
*   🗄️ **MySQL 8.x**
*   🏗️ **Maven 3.x**

### ⚙️ Quick Setup
1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/Yogaraj02/EL-Spring-Boot-Application.git
    ```
2.  **Database Configuration**:
    Create a DB named `college_events_db` and update `application.properties`:
    ```properties
    spring.datasource.url=jdbc:mysql://localhost:3306/college_events_db
    spring.datasource.username=your_username
    spring.datasource.password=your_password
    ```
3.  **Launch**:
    ```bash
    mvn spring-boot:run
    ```
